# Conditionals

## if

```text
if (<expression>) {
    <commands>
}
```

## else

```text

if (<expression>) {
    <commands>
} else {
    <commands>
}
```

## elsif

```text

if (<expression>) {
    <commands>
} elif (<expression>) {
    <commands>
} elif (<expression>) {
    <commands>
} elif (<expression>) {
    <commands>
} else {
    <commands>
}
```

## Ternary if

```text
<variable or constant> = <expression> ? <command 1> : <command 2>
```

## Switch

```text
switch(<expression>) {
    case <value>:
        <commands>
        break
    case <value>:
        <commands>
        break
    case <value>:
        <commands>
        break
    default:
        <commands>
        break
}
```
