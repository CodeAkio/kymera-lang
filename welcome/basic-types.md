# Basic types



| Type | Description | Range | Default Value |
| :--- | :--- | :--- | :--- |
| uint8 / ubyte | 8-bit unsigned integer type | 0 to 255 | 0 |
| int8 / byte | 8-bit signed integer type | -128 to 127 | 0 |
| int16 | 16-bit signed integer type | -32,768 to 32,767 | 0 |
| uint16 | 16-bit unsigned integer type | 0 to 65,535 | 0 |
| int32 / int | 32-bit signed integer type | -2,147,483,648 to 2,147,483,647 | 0 |
| uint32 / uint | 32-bit unsigned integer type | 0 to 4,294,967,295 | 0 |
| int64 | 64-bit signed integer type | -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807 | 0L |
| uint64 | 64-bit unsigned integer type | 0 to 18,446,744,073,709,551,615 | 0L |
| decimal | 128-bit precise decimal values with 28-29 significant digits | \(+ or -\)1.0 x 10e-28 to 7.9 x 10e28 | 0M |
| float32 | 32-bit single-precision floating point type | -3.4 x 1038 to + 3.4 x 1038 | 0.0F |
| float64 | 64-bit double-precision floating point type | \(+/-\)5.0 x 10-324 to \(+/-\)1.7 x 10308 | 0.0D |
| char | 16-bit Unicode character | U +0000 to U +ffff | '\0' |
| bool | Boolean value | true or false | false |
