# Comparison Operators

## Igual

```ruby
> 10 == '10'
=> false

> 10 == 10.0
=> false

> 10 == 10
=> true
```

## Diferente

```python
$ 10 != '10'
# true

$ 10 != 10.0
# true

$ 10 != 10
# false
```

## Maior

```ruby
$ 10 > 10
# false

$ 100 > 10
# true

$ 10.0 > 10
# false

$ 'b' > 'a'
# true
```

## Menor

```python
$ 10 < 10
# false

$ 100 < 10
# true

$ 10.0 < 10
# false

$ 'b' < 'a'
# true
```
