# Arithmetic Operators

## Soma

```python
1 + 2
# 3

10 + 2.2
# 12.2

"$ " + "10.90"
# "$ 10.90"

"$ " + 10.90
# "$ 10.90"

[1, 2, 3] + [4]
# [1, 2, 3, 4]

t[1, 2, 3] + t[4]
# t[1, 2, 3, 4]

{ name: "Pedro" } + { age: 22 }
# { name: "Pedro", age: 22 }
```

## Subtração

```python
1 - 2
# -1

10 - 2.2
# 7.8

[1, 2, 2, 3] - [2]
# [1, 3]

t[1, 2, 3] - t[2]
# t[1, 3]

{ name: "Pedro", age: 22 } - { age: 22 }
# { name: "Pedro" }
```

## Multiplicação

```python
2 * 2
# 4

10 * 0.5
# 5.0

"-" * 5
# "-----"

[1, 2, 3] * 2
# [1, 2, 3, 1, 2, 3]
```

## Divisão

```python
8 / 2
# 4

5 / 2.5
# 2.0
```

## Resto

```python
8 % 2
# 0

22.0 % 4
# 2.0
```

Ou de forma verbosa

```python
8 mod 2
# 0
```

## Potência

```python
2 ** 4
# 16

2 ** 8.0
# 256.0
```

## Flor Division

```python
7 // 3
# 2

7 // 4.0
# 1
```

## Increment

```nim
var num = 1
nim.inc
# 2
```

## Decrement

```nim
var num = 1
nim.dec
# 0
```
