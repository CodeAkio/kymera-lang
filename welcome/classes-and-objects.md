## Sintaxe

```kotlin
class Pessoa {
    nome string
    idade int

    _init(nome string, idade int) {
        this.nome = nome
        this.idade = idade
    }
    
    fun ola() {
        writeln("Olá ${this.nome}")
    }
}
```
